# @whisper-cpp-node/win32-x64

Native binary for Windows x64.

This package is automatically installed by `@whisper-cpp-node/core` on compatible platforms.

**Do not install this package directly.** Install `@whisper-cpp-node/core` instead.

## Requirements

- Windows 10/11 x64
- Node.js 18+

## GPU Acceleration

This binary is built with Vulkan support for GPU acceleration. If you have a Vulkan-capable GPU, GPU acceleration is enabled by default (`use_gpu: true`).
